Online Chess Game

Project Foundation

This project was based on an open-source Unity chess implementation. My work focused on improving multiplayer functionality, synchronization, and overall game reliability, especially for WebGL deployment.

Turn System & Game Logic

  • Implemented full turn synchronization between clients.
  • Ensured consistent board state replication.
  • Developed a system to display captured pieces on the side of the board.
  • Implemented match timers to determine victory and defeat conditions.

Online Integration & Networking

Although the original project already included an online multiplayer system, it relied on Unity’s legacy networking solution, which presented stability issues, especially for WebGL builds.

I migrated the project to Unity Netcode, refactoring networking logic to properly replicate chess pieces and synchronize game state across clients in a reliable and scalable way.

This migration significantly improved multiplayer stability, particularly for browser-based WebGL gameplay.
