Route Raiders – Full Stack Adventure Platform

I independently engineered this extensive location-based adventure game and platform, handling all aspects of development across backend, frontend, game logic, networking, and infrastructure. The project aimed to turn real-world exploration into a gamified cultural experience. :contentReference[oaicite:0]{index=0}

Full System Development

I was solely responsible for the full stack development:

  • .NET Core backend with secure APIs
  • Angular CMS for content, quests, collectibles, and QR rewards management
  • Unity client with Mapbox integration for real-time location gameplay
  • JWT-based authentication with secure login and session handling

Gameplay Features & Integrations

The game combines geolocation, dynamic quests, and player engagement:

  • Real-time player location tracking and positioning on Mapbox maps
  • Route missions with narrative content tied to specific points of interest
  • Interactive collectibles and environmental items managed via CMS
  • Daily loot mechanics: server generates treasure chests around player location
  • Partner-managed voucher challenges with QR code scanning for rewards

CMS & Tools for Content Team

I developed a custom CMS tool using Angular and TypeScript that allowed the content team to author:

  • Dialogue trees with branching responses and image uploads
  • Locations, route descriptions, and educational text
  • Quest and collectible configurations
  • Voucher/QR code rewards and localization metadata

DevOps & Deployment

I also took care of infrastructure and deployment:

  • Configured HTTPS and secure hosting via NGINX
  • Set up automated builds and deployments using Jenkins
  • Ensured end-to-end secure communication across services

Impact & Scope

This project transformed city exploration into a gamified adventure, integrating educational content, location-based missions, interactive storytelling, and gamified mechanics.
