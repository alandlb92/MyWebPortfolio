Architecture & Systems

The game was structured with a modular gameplay architecture, separating board logic, input handling, animation flow, and UI systems to ensure maintainability and scalability. Core gameplay mechanics were implemented using clean C# patterns, with a focus on readability and performance.

Custom Level Editor Tool

To streamline content production, I developed a custom Unity Editor tool for creating and configuring levels. This tool allowed visual board layout creation, validation of move rules, and quick iteration without manually editing scene data. The system significantly improved workflow efficiency and reduced human error during level design.

Animation System

All gameplay animations were implemented programmatically using Animation Curves and Coroutines. Instead of relying solely on Unity's Animator, I built a lightweight animation flow system to control timing, movement interpolation, scaling effects, and feedback transitions. This approach provided precise control over animation sequencing and responsiveness.

UI & Feedback

The entire UI system was developed from scratch, including state transitions, animated panels, and visual feedback systems. UI animations were also driven by Curves and Coroutines, ensuring consistency between gameplay and interface motion design.

Mobile Integration

The project was deployed for Android, including full integration of mobile monetization systems. I implemented advertisement services and in-app purchases (IAP), handling platform-specific configuration, purchase validation flow, and user feedback for transaction states.

Key Technical Highlights

  • Unity (C#) gameplay architecture
  • Custom Unity Editor tooling
  • Programmatic animation system using Animation Curves & Coroutines
  • Modular board logic and rule validation
  • Custom UI system with animated transitions
  • Android deployment with Ads & In-App Purchases integration
