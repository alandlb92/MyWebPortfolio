Builder Scenes – Real-Time Content Creation Tool

Builder Scenes was a Unity-based content creation software developed for Creative Drive (now Accenture). The tool allowed users to build and customize fully configurable 3D environments, which were later processed by the platform to generate photorealistic renders using VRay.

Core Functionality

The application functioned as an in-game scene editor, where users could:

  • Configure walls, ceilings, floors, and surface materials
  • Apply different coverings and finishes
  • Adjust cameras, including field of view and focus settings
  • Download and place platform-managed assets dynamically into the scene

Once a scene was fully configured, it was uploaded to the platform, where high-quality photorealistic images were generated using VRay.

My Contributions

  • Improved and refactored the in-game hierarchical interface, designing a tree-view system similar to Unity’s Hierarchy panel for managing scene objects.
  • Implemented a real-time focus simulation system to preview depth-of-field effects that would later be rendered in VRay.
  • Worked on performance optimizations across the application.
  • Contributed to the asset download and integration system, ensuring assets were properly fetched, parsed, and instantiated.
  • Improved scene object manipulation and movement systems.

Technical Scope

The project required strong integration between Unity runtime systems and an external rendering pipeline, bridging real-time scene assembly with offline photorealistic rendering.
