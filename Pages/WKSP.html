WKSp – 2D Fighting Game

WKSp was a 2D fighting game developed around employees from the São Paulo branch of Wieden+Kennedy. I was solely responsible for the full engineering of the project, handling gameplay systems, tools, automation, AI, and combat logic.

Scalable Character System

With over 100 playable characters and a one-year production scope, scalability was critical. To achieve this, I designed an archetype-based character system that allowed shared behaviors with controlled variations.

  • Created a reusable archetype framework
  • Implemented variations based on employee roles
  • Designed unique special abilities per character category

This approach enabled efficient production while maintaining gameplay diversity.

Automated Asset Pipeline

To support production speed, I developed an automated asset management system using .NET Core, Angular, and SQL.

  • Pixel artists uploaded animation frames, portraits, and character data via a web interface
  • Assets were stored and organized on the server
  • The Unity client dynamically downloaded and integrated the content

This automation was fundamental to delivering the project on time.

Combat System & Collision Architecture

I implemented a detailed 2D combat collision system with three distinct collider types:

  • Hit Collision – Detects when attacks connect
  • Damage Collision – Applies damage logic
  • Physics Collision – Handles character physical interactions

Special attention was given to animation timing. Damage boxes and hit detection were precisely aligned with specific animation frames to ensure accurate and responsive combat.

I also implemented combat feedback systems such as hit freeze (frame pause) and screen shake to enhance impact and player feedback.

AI & Game Systems

  • Developed AI behavior using a State Machine architecture
  • Implemented full game flow for 1v1 and Story Mode
  • Built character selection and stage selection interfaces
  • Handled all UI and gameplay flow logic

The project required careful system design to support scale, maintain responsiveness, and ensure production efficiency.
